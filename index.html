    What is HelpChat?

    Help chat is a plugin aimed towards helping users grab the attention of online admins.

    How do I install and set-up HelpChat?

    To install HelpChat all you need to do is drop it in your bukkit plugins folder and reload/restart your server. There is no configuration file for this at time but i might ad an option on customising the commands and the colour of the chat messages.
